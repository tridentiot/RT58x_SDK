◆ sys_timer_create()

sys_timer_t* sys_timer_create ( const char *  name,
uint32_t  period,
uint32_t  auto_reload,
sys_timer_pri_t  priority,
uint32_t  execute_mode,
void *  cb_param,
sys_timer_cb  cb_function 
)

Create a software timer.

Parameters
nameA human readable text name that is assigned to the timer.
periodThe period of the timer.
auto_reloadIf auto_reload is set to TRUE, then the timer will expire repeatedly with a frequency set by the period parameter. If auto_reload is set to FALSE, then the timer will be a one-shot and enter the dormant state after it expires.
priorityThe priority of the timer.
execute_modeThe execute mode of the timer.
cb_paramThe input parameter of the call back function.
cb_functionThe function to call when the timer expires.
Returns
sys_timer_t* If the timer is created successfully then a handle to the newly created timer is returned. If the timer cannot be created the NULL is returned.
