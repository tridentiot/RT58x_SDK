uint32_t create_dma_link_list ( dma_linklist_request_t *  dma_linklist_request,
uint32_t  entry_number,
dma_linklist_entry_t *  DMA_link_list,
uint32_t  control_mode 
)

Use create_dma_link_list function to create hardware descriptor for dma scatter move.

Parameters
dma_linklist_requestpointer to dma_linklist_request_t that software describe data move "instructions"
entry_numbernumber of linklist requests in the dma_linklist_request array
dma_link_listpointer to dma_linklist_entry_t array that will save hardware descriptor setting.
control_modedma transfer mode

dma_linklist_request is the "dma move instruction" array which has entry_number item. And DMA_link_list is the link_list entry used by dma hardware. It should have entry_number item space too.
