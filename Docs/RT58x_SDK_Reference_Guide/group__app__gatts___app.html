Get GATTS Handle Numbers

Attention
- role = BLE_GATT_ROLE_CLIENT
MUST call this API to get service information after received BLECMD_EVENT_ATT_DATABASE_PARSING_FINISHED
  • role = BLE_GATT_ROLE_SERVER
    MUST call this API to get service information before connection established.
Parameters
[in]host_id: the link's host id.
[out]p_info: a pointer to GATTS information.
Return values
BLE_ERR_INVALID_HOST_ID: Error host id.
BLE_ERR_INVALID_PARAMETER: Invalid parameter.
BLE_ERR_OK: Setting success.

GATTS Initialization

Attention
There is only one instance of GATTS shall be exposed on a device (if role is BLE_GATT_ROLE_SERVER).
Callback shall be ignored if role is BLE_GATT_ROLE_SERVER).
Parameters
[in]host_id: the link's host id.
[in]role: BLE GATT role.
[in]p_info: a pointer to GATTS information.
[in]callback: a pointer to a callback function that receive the service events.
Return values
BLE_ERR_INVALID_HOST_ID: Error host id.
BLE_ERR_INVALID_PARAMETER: Invalid parameter.
BLE_ERR_CMD_NOT_SUPPORTED: Registered services buffer full.
BLE_ERR_OK: Setting success.
